# Техническое задание

> Проект M1: Предсказание риска сердечного приступа

---

## 1. Цель проекта

Разработать модель машинного обучения для предсказания риска сердечного приступа, а также подготовить библиотеку и интерфейс для предсказаний на тестовой выборке.

---

- [x] Исследование датасета (EDA)

- [x] Обучение модели
- [x] Подготовка предсказания на тестовой выборке (`id`, `prediction`)
- [x] Подготовка скриптов и библиотеки для обработки данных
- [x] Написание инструмента для тестирования (веб-интерфейс)
- [x] Оформление документации

---

## 3. Описание данных

**Источник:** Данные пациентов для предсказания риска сердечных приступов

**Признаки:**
- `id` — идентификатор записи
- Антропометрические параметры (возраст, вес, рост, BMI)
- Привычки (курение, качество сна, алкоголь)
- Давление (систолическое, диастолическое)
- Хронические заболевания (диабет, ожирение)
- Биохимия крови (холестерин, триглицериды, тропонин)

**Целевая переменная:** Высокий/низкий риск сердечного приступа (бинарная классификация)

---

## 4. Метрика качества

**Выбранная метрика:** F1-score

**Обоснование:** При несбалансированных классах F1-score учитывает как точность (precision), так и полноту (recall), что важно для медицинских данных.

**Целевое значение:** F1 > 0.49
